Picada

A la hora de armar una picada, es importante tener en cuenta que tipo de fiambres, frutas o vegetales usar. Se recomienda utilizar fiambres o quesos con bajo contendio calorico que aporten algo de proteina y que su nivel graso no sea tan elevado. Aqui el objetivo es buscar comidas voluminosas que logren crear esa sensacion de saciedad.

Pizza

El objetivo con este tipo de comidas es la creatividad. Las variaciones en los ingredientes que uno use puede adaptar la pizza a distintas necesidades. El uso de vegetales como toppings brindara mayor micronutrientes, el uso de alguna carne desmenusada aportara proteina. El objetivo aqui es usar la imaginacion y crear.

Revuelto de vegetales

Esta opcion propociona un gran volumen con un bajo aporte calorico.Rica en vitaminas A, C y Calcio. Baja en grasas con un buena distribucion de carbohidratos y proteinas. Gran opcion para vegetarianos y veganos.

Tostadas

Gran opcion para desayuno o colacion. Comida de rapida absorcion con una buena distribucion de carbohidratos, grasas y proteinas. Ideal para consumir previo a realizar una actividad fisica.
